# 4.4 명령어 Fast-Forward merge, 3-way merge, rebase

### 4.4.1 Fast-Forward
Fast-Forward 깃의 가장 간단한 브랜치 병합방식이며 풀어서 쓰면 **빨리감기**라고 할 수 있다.


    $ git merged 브랜치이름
merge 명령어는 **현재 브랜치를 기준**으로 다른 브랜치의 모든 커밋을 병합한다.

Fast-Forward의 예시를 간단하게 실행해보겠다.
